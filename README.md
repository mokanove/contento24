# contento24
一个开源且隐私的24小时在线WebP2P聊天室

> 运行方法
>
> 需要先把npm nodejs git ssh全家都搬上来
```
git clone git@github.com:mokanove/contento24.git
pnpm install
#↑或者随便什么npm
node server.js
```
> 当看到
```
=================================================
🚀 世界光明网全双工聊天室已成功发射！
端口: 3000
内网测试: http://localhost:3000
外网测试: 请直接通过你的IP地址/域名:端口访问
=================================================
```
> 说明跑起来了
